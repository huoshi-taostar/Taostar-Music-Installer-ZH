## 📖 项目简介

**桃星音乐 (Taostar Music)** 是一款以**放松、治愈、沉浸体验**为核心设计的音乐播放器。

通过融合：

* 音乐
* 动态壁纸
* 自然环境音
* 可视化效果

为用户打造一个安静、舒适、富有氛围感的使用环境。

---

## 核心功能

* 支持添加并播放本地音乐
* 高质量动态背景（动漫风景 / 自然风光）
* 自然音效（雨声 / 海浪 / 风声等）可叠加播放
* 音量独立控制（音乐 + 环境音）
* 音乐可视化（频谱动画效果）
* 轻量级设计，运行流畅不卡顿
